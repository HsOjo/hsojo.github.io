麒麟系统瘦身

2022-05-25  683 字  阅读时间 3 分钟

前言

近期在完成项目部署的过程中，遇到了一台特殊设备。该设备在安装Ubuntu Server系统后，无法驱动网卡；经过一番分析，发现是Linux内核没有集成相关驱动。

不过由于麒麟系统是由Ubuntu衍生的，用来部署项目也不是不可，但要用桌面系统来部署项目，就得精简一下了。

关闭应用安装验证

sed -i 's/allow-kylinsign/#allow-kylinsign/g' /etc/dpkg/dpkg.cfg
sed -i 's/verify-kylinsign/#verify-kylinsign/g' /etc/dpkg/dpkg.cfg

禁用KYSEC

KYSEC是麒麟系统的安全防护模块，会在启动程序时进行拦截，一定程度上会降低系统性能，禁用之。

修改文件/etc/default/grub，将GRUB_CMDLINE_LINUX_DEFAULT中的security=kysec修改为security=即可。

命令方式

sed -i 's/security=kysec/security= /g' /etc/default/grub
  • 执行以下命令以更新引导
# 标记为手动安装，否则后续自动卸载依赖会导致出错apt install archdetect-deb
setstatus disable
update-grub
  • 在完成后续操作后，重启即可

卸载常驻服务

apt purge -y samba xrdp cups-daemon xl2tpd

卸载桌面环境

  • 可选操作：卸载麒麟全家桶（除apt所依赖的kylin-keyring）
apt purge -y kylin-activation kylin-background-upgrade kylin-burner kylin-ca-certificates kylin-calculator kylin-camera kylin-display-switch kylin-fonts kylin-installer kylin-ipmsg kylin-kmre-daemon kylin-kmre-display-control kylin-kmre-image-data kylin-kmre-manager kylin-kmre-window kylin-log-viewer kylin-music kylin-nm kylin-photo-viewer kylin-printer kylin-recorder kylin-scanner kylin-screenshot kylin-service-support kylin-signtool kylin-software-center kylin-software-properties kylin-sso-client kylin-update-desktop-config kylin-update-desktop-security-database kylin-update-manager kylin-usb-creator kylin-user-guide-common kylin-user-guide kylin-verify-gui kylin-verify kylin-video kylin-xinput-calibrator
  • 卸载国产特色软件（需要手动清扫/opt）
apt purge -y browser360-cn-stable wps-office sc-reader sogouimebs aurora.a3-mfp browser360-cn-stable certaide-kylin hplip
  • 卸载桌面管理器
apt-get purge -y lightdm ukui*apt autoremove -y
  • 配置SystemD
systemctl set-default multi-user.target
systemctl isolate multi-user.target
