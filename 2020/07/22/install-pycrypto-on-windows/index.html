这个错误与以前转的博客几乎一致。

首先安装VC++ Build Tools，然后执行以下命令指定CL环境变量。

export CL="/FI \"$VS140COMNTOOLS..\\..\\VC\\INCLUDE\\stdint.h\""

然后就可以正常通过pip安装PyCrypto了。
